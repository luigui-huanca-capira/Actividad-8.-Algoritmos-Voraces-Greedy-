# Actividad 8 - Algoritmos Voraces (Greedy)

## Descripción
Este proyecto contiene la solución en Java de la Actividad 8 de Algoritmos Voraces:

1. **Mochila Fraccional**
2. **Cobertura de Antenas**

El archivo principal es:

- `Actividad8Greedy.java`

## Requisitos
- Java JDK 8 o superior
- Git (para clonar/versionar)

## Cómo compilar y ejecutar

En la carpeta del proyecto:

```bash
javac Actividad8Greedy.java
java Actividad8Greedy
```

## Salida esperada
El programa imprime los resultados de los ejemplos guía para ambos problemas:
- Mochila Fraccional (2 ejemplos)
- Cobertura de Antenas (2 ejemplos)
